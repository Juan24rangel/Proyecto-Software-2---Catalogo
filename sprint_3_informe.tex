\section{Objetivo del Sprint 3}

\begin{infobox}[Objetivo General]
El objetivo del Sprint 3 es expandir el Microservicio de Cat\'alogo con soporte para categor\'ias, validaci\'on de im\'agenes y una interfaz gRPC de bajo latency. Se implementan las historias Should Have pendientes (HU-07, HU-08, HU-09) para consolidar un cat\'alogo multi-capa con organizaci\'on jer\'arquica de categor\'ias, validaci\'on de activos multimedia y comunicaci\'on inter-microservicios.
\end{infobox}

\subsection{HU-08 --- Validaci\'on de Im\'agenes}

Se cre\'o el servicio \texttt{IImageValidator} que valida:

\begin{itemize}[leftmargin=1.5em]
  \item \textbf{Formato URL:} Debe ser \'URL v\'alida (http/https).
  \item \textbf{Extensiones permitidas:} \texttt{.jpg, .jpeg, .png, .gif, .webp, .svg}.
  \item \textbf{Cantidad:} M\'inimo 1, m\'aximo 5 im\'agenes por producto.
  \item \textbf{Validaci\'on case-insensitive:} Las extensiones se validan en min\'usculas.
\end{itemize}

\codeblock{M\'etodo ValidateImageUrls en ProductsController}{%
if (request.Images != null \&\& request.Images.Count > 0)\\
\{\\
\quad var (isValid, errorMessage) = \_imageValidator.ValidateImageUrls(\\
\quad\quad request.Images, minImages: 1, maxImages: 5);\\
\quad if (!isValid)\\
\quad\quad return BadRequest(new \{ message = errorMessage \});\\
\}%
}

\subsection{HU-09 --- Integraci\'on gRPC}

\subsubsection{Contrato .proto}

Se define en \texttt{Protos/catalog.proto}:

\codeblock{Definici\'on del servicio gRPC}{%
service CatalogService \{\\
\quad rpc GetProduct(GetProductRequest) returns (ProductResponse);\\
\quad rpc GetProducts(GetProductsRequest) returns (GetProductsResponse);\\
\quad rpc GetProductsByIds(GetProductsByIdsRequest)\\
\quad\quad\quad\quad returns (GetProductsResponse);\\
\}%
}

\subsubsection{M\'etodos gRPC Implementados}

\begin{itemize}[leftmargin=1.5em]
  \item \textbf{\texttt{GetProduct(id)}:} Obtiene producto por ID. Responde <100ms.
  \item \textbf{\texttt{GetProducts(page, pageSize, search)}:} Listado paginado con b\'usqueda opcional.
  \item \textbf{\texttt{GetProductsByIds(ids)}:} Obtiene m\'ultiples productos por lista de IDs.
\end{itemize}

\subsubsection{Error Handling en gRPC}
\begin{itemize}[leftmargin=1.5em]
  \item \texttt{StatusCode.InvalidArgument:} Entrada inválida.
  \item \texttt{StatusCode.NotFound:} Producto/recurso no encontrado.
  \item \texttt{StatusCode.Internal:} Error interno del servidor.
\end{itemize}
